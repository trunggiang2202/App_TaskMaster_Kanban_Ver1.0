# **App Name**: TaskMaster Kanban

## Core Features:

- Task Creation Panel: A left sidebar panel allowing users to quickly add new tasks with a button.
- Task Setup Modal: A modal that appears when the 'Add Task' button is clicked, allowing detailed setup of tasks.
- Parent Task with Deadline: Ability to set a deadline for the parent task, including a progress bar that counts down towards the deadline.
- Subtask Management: Functionality to add subtasks to a parent task.
- Subtask Deadlines and Start Dates: Each subtask has its own deadline and start date, influencing the parent task's overall progress.
- Task Completion Logic: The parent task is only marked as complete when all subtasks are finished.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey a sense of stability and focus.
- Background color: Very light grey (#F0F0F5) to keep the interface clean and unobtrusive.
- Accent color: Bright orange (#FF9800) to highlight key actions and deadlines.
- Body and headline font: 'Inter', a sans-serif, providing a modern and neutral look.
- Use simple, geometric icons for task categories and status indicators.
- Kanban board layout with drag-and-drop functionality for tasks.
- Subtle animations for task transitions and progress updates.
